# Smart Expense Tracker

A simple Python-based Smart Expense Tracker to record, manage, and analyze daily expenses.

## Features

- Add daily expenses
- View recorded expenses
- Calculate total expenses
- Simple menu-driven interface
- Handles invalid amount input

## Technologies Used

- Python

## How to Run

1. Clone the repository.
2. Open the project folder.
3. Run the following command:

python expense_tracker.py

## Project Purpose

This project was developed to practice Python fundamentals such as functions, lists, dictionaries, loops, and exception handling.
